Formulario de Registro

Favor completar las siguientes solicitudes:

Nombres

Debe contener mínimo 2 letras, puede contener espacios, no puede contener caracteres especiales.

Correo Electrónico

El correo debe tener el formato tradicional

Teléfono

El teléfono debe iniciar con el prefijo internacional y tener minimo 12 caracteres y maximo 20.

Contraseña

La contraseña debe tener minimo 8 caracteres

Confirmar Contraseña

La contraseña debe coincidir con la confirmación

Pasaporte

El número de pasaporte debe coincidir con el formato internacional
